# NewGenLearning

**Rethinking Code Assessment for the AI Era**

We're testing whether people can *write* code — but in an age where AI generates it, the real skill is knowing whether code is *good*. NewGenLearning is a platform that assesses what actually matters: the ability to evaluate, reason about, and distinguish between code solutions.

## The Problem

### How Code Is Actually Written

The development workflow hasn't changed: write code, commit, push, merge, deploy. What *has* changed is who writes it — increasingly, it's AI. The critical skill is no longer *writing* code, but **discerning** good code from bad. Think of it like this: you don't need to be a great writer to be a great editor.

### How Code Is Currently Assessed

**In Education**
- Teach a concept, build something that runs, get a grade
- True/False and multiple-choice questions that AI can answer trivially
- Stays at surface-level concept verification

**In Hiring (HR)**
- LeetCode-style: given a problem, write code, pass test cases
- No way to tell if someone truly understands the concept or just got lucky
- Tests code *writing* ability, but the real need is code *judgment*

### The Gap

We're in an era with calculators, but still testing who can do arithmetic by hand. AI generates code, yet assessment frameworks haven't caught up. Both schools and companies want to find people who truly *understand* code — but current methods can't identify them.

## Our Solution

### Reasoning-First Assessment

Instead of "write the code," we ask: "which solution is better, and *why*?"

- Present a real-world scenario with multiple code solutions
- Require written **reasoning** for every choice
- Evaluate the *thought process*, not just the answer

### AI-Powered Validation

AI acts as both a **validator** and an **instructor assistant**:

- **For Education**: Highlights each student's knowledge gaps and generates targeted feedback recommendations for instructors — "This student needs reinforcement on X"
- **For Hiring**: Distinguishes candidates who genuinely understand concepts from those who guessed correctly. Can also detect AI-generated responses

### Why It's Better

- Evaluates **thinking**, not just output
- Provides **actionable insights** for instructors and hiring managers
- Assesses **documentation ability** as a side effect (through written reasoning)
- Built for an era where code generation is commoditized but code judgment is not
